# SersmickWeatherApp

A weather application that displays weather information in a responsive web interface.

## Getting Started

### Prerequisites
- Node.js installed on your machine

### Installation & Running Locally

#### Option 1: Using Windows Command (Easiest)
1. Open Command Prompt or PowerShell
2. Navigate to the project folder:
   ```
   cd path\to\SersmickWeatherApp
   ```
3. Run the included batch script:
   ```
   start-local-server.cmd
   ```
4. Open your browser and go to: **http://localhost:4173**

#### Option 2: Using Node.js Directly
1. Open Command Prompt or PowerShell
2. Navigate to the project folder:
   ```
   cd path\to\SersmickWeatherApp
   ```
3. Start the server:
   ```
   node server.js
   ```
4. Open your browser and go to: **http://localhost:4173**

## Hosting on GitHub

### 1. Initialize Git Repository (if not already done)
```bash
git init
git add .
git commit -m "Initial commit - SersmickWeatherApp"
```

### 2. Create Repository on GitHub
- Go to [github.com/new](https://github.com/new)
- Create a new repository named `SersmickWeatherApp`
- Do **NOT** initialize with README, .gitignore, or license

### 3. Push to GitHub
Replace `YOUR_USERNAME` with your GitHub username:

```bash
git remote add origin https://github.com/YOUR_USERNAME/SersmickWeatherApp.git
git branch -M main
git push -u origin main
```

### 4. View Your App
Once pushed to GitHub, you can:
- View the repository: `https://github.com/mafari10/SersmickWeatherApp`
- Clone it locally: `git clone https://github.com/mafari10/SersmickWeatherApp`

## Deploying to the Web

### Option 1: Using GitHub Pages (Static Hosting)
GitHub Pages requires the repository to be public. Your app can be deployed for free at `https://mafari10.github.io/SersmickWeatherApp/`

1. Go to your GitHub repository Settings → Pages
2. Select "Deploy from a branch" 
3. Choose `main` branch and save
4. Your app will be available at the URL provided
